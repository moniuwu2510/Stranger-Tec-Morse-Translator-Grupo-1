
 Stranger-Tec-Morse-Translator-Grupo-1
En este repositorio se encuentra el proyecto 1 de fundamentos de sistemas de la escuela de ingeniería en computadores del Tecnológico de Costa Rica
Somos el grupo 1 formado por los estudiantes Monica Blanco Jiménez, Alexander Rojas Lai y Jehudy Sanchez Gomez
En el siguiente repositorio usted encontrará el código de la GUI solicitada, el código para que la maqueta con la raspberry funcione, la documentación necesaria para realizar la instalación y correr los códigos así como las librerías de tkinter que puede requerir en su equipo para que funcione, y por ultimo el API que servirá de intermediario para que la GPIO tenga conexión tanto con la maqueta como con la interfaz gráfica

Requiere installar pygame para manejar el sonido
use:
pip install pygame
